# Canon Interview

Questions to populate your `Canon/` files.

**How to use this:** Work through each section in order. Answer directly below each question.
Don't edit for polish — write like you're explaining it to a smart new hire on their first day.
Honest and plain beats articulate and vague every time.

When you're done, save this file and type `build canon from interview` in Claude Code.
Claude will read your answers and populate all five Canon files.

**Supplementing with exports:** If you have exports from existing tools (listed throughout
each section), drop them into `Raw/canon-source/` alongside this file. Claude will
cross-reference your answers with the exports and flag any gaps or contradictions.

**Time estimate:** 45–90 minutes for a thorough pass. You can return to incomplete
sections — leave `[SKIP]` as a placeholder and Claude will note what's missing.
