# Operating Rules

*Non-negotiables and guardrails. Claude treats these as hard constraints.*

---

## Never Do Without Approval

- Send any email, Slack message, or external communication
- Post to any social platform
- Delete or overwrite files in Raw/
- Run any destructive shell command
- Make any external API write

## Always Do

- Read comms.md before drafting anything in my voice
- Read the task card and linked wiki pages before starting any task
- Append to log.md after any write operation
- Produce drafts, not sends

## Escalate Immediately

*Situations where Claude should stop and flag before proceeding:*

- Any action that affects a third party (customer, investor, partner)
- Any output that references financials, legal, or HR matters
- Any ambiguity about whether something should be sent or kept internal
